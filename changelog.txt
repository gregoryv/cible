# Changelog

All major changes will be documented in this file.

## [0.4.0] - 2022-05-08

- Notify others when character is nearby or goes away
- Tidy tile output
- Simplify creating areas and linking tiles
- Add navigation in ne, se, sw and nw directions
- Initial area is a large stateroom
- Display help in a framed box

## [0.3.0] - 2022-05-07

- Add EventDisconnect, currently a noop
- Set player name to environment variable USER
- Include short description when looking around
- Remove colors

## [0.2.0] - 2022-05-06

- Show tile desription when moving around
- Warn when trying to move in wrong direction
- Include exits when looking around

- Issue: no colors in windows terminal

## [0.1.0] - 2022-05-04

- Basic client server in place
- Characters can move around using n,e,s,w keys
- Look around using l
- Quit using q
